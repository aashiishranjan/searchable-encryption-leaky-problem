# Searchable Encryption Leaky Problem Project

This web application provides a platform for analyzing searchable encryption with a focus on its leaky problem. The project is structured into a backend using Flask and a frontend using React.

## Structure

- `backend/`: Flask backend consisting of REST API endpoints.
- `frontend/`: React frontend for user interaction.

### Features
1. Interactive dashboard for Searchable Encryption analysis
2. Document upload and management
3. Keyword search with BEFORE/AFTER mode toggle
4. Complete 6-step process display
5. Leakage visualization charts
6. Query history tracking
7. Professional, responsive UI
